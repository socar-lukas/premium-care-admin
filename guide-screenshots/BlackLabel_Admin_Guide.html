1. 소개

BlackLabel Admin은 차량 관리 현장에서 사용하는 점검 기록 시스템입니다.

주요 기능

  • 반납상태 등록 - 고객 반납 시 차량 상태 기록
  • 세차·점검 등록 - 세차 후 점검 완료 기록
  • 소모품·경정비 등록 - 소모품 교체, 경정비, 사고수리 기록
  • 차량 문열기 - 옥스트라 바로가기

접속 방법

웹 브라우저에서 blacklabel.vercel.app에 접속합니다.

TIP: 스마트폰 홈 화면에 추가하면 앱처럼 편리하게 사용할 수 있습니다.

2. 대시보드 (메인 화면)

화면 구성

1 상단 통계 카드
  • 운행중 - 현재 고객이 이용 중인 차량 수
  • D+1 예약 - 24시간 내 예약 시작되는 차량 수
  • 점검필요 - 세차·점검이 필요한 차량 수 (9시간 내 세차점검 기록 없음)
2 액션 버튼
  • 반납상태 등록 - 고객 반납 시 상태 기록
  • 세차·점검 등록 - 세차 후 점검 완료 기록
  • 소모품·경정비 - 소모품/경정비/사고수리 기록
3 차량 목록
  • 등록된 모든 차량을 확인할 수 있습니다
  • 차량을 클릭하면 상세 페이지로 이동합니다
  • 상태 아이콘으로 운행중/대기중/점검필요 여부를 확인할 수 있습니다

3. 반납상태 등록

고객이 차량을 반납했을 때 차량 상태를 기록합니다.

입력 항목

1 차량번호 선택

차량번호 4자리 이상 입력하면 자동으로 검색됩니다.

2 담당자 선택

본인 이름을 선택하거나 새로 입력합니다.

3 차량 반납 상태 점검
항목 선택값
반납차량 오염도 상 / 중 / 하
외관이상 유형 이상없음 / 경미(운행가능) / 파손(운행불가)
타이어 상태 각 타이어별: 정상 / 경미 / 심각
내부 오염 위치 운전석, 보조석, 뒷좌석, 트렁크 (복수 선택)
4 점검 전 외관 사진

전면 / 우측 / 후면 / 좌측 - 4방향 모두 촬영해야 합니다.

5 점검 전 내부 사진

컵홀더, 발매트 등 오염 부위를 촬영합니다.

주의: 모든 외관 사진(전/우/후/좌)과 내부 사진을 등록해야 저장 버튼이 활성화됩니다.

4. 세차·점검 등록

세차 완료 후 점검 결과를 기록합니다.

입력 항목

1 조치 사항
항목 선택값
세차 정밀세차 / 약식세차
배터리 상태 정상 / 전압낮음 / 충전필요 / 교체필요
와이퍼/워셔액 와이퍼 불량, 워셔액 부족 (복수 선택)
경고등 엔진, 배터리, 타이어, 기타 (복수 선택)
2 점검 후 외관 사진

세차 완료 후 전면 / 우측 / 후면 / 좌측 4방향 촬영

3 점검 후 내부 사진

세차 완료 후 실내 상태 촬영

TIP: 세차점검 기록이 있어야 "점검필요" 상태가 해제됩니다.

5. 소모품·경정비 등록

소모품 교체, 경정비, 사고수리 작업을 기록합니다.

작업 항목

1 소모품 교체

엔진오일, 에어컨필터, 타이어, 배터리, 와이퍼, 브레이크패드 등

2 경정비

얼라인먼트, 광택/코팅, 흠집제거, 덴트수리, 내부크리닝, 시트수리, 실내소독 등

3 사고수리

범퍼, 휀다, 도어, 보닛, 트렁크, 루프, 미러, 유리, 라이트, 휠 등

6. 차량 상세보기

차량을 클릭하면 상세 페이지에서 다음 정보를 확인할 수 있습니다.

주요 기능

1 차량 문열기

상단의 "문열기" 버튼을 클릭하면 옥스트라로 바로 이동합니다.

2 반납상태 기록

해당 차량의 반납상태 기록 목록을 확인할 수 있습니다.

3 세차·점검 기록

해당 차량의 세차·점검 기록 목록을 확인할 수 있습니다.

당월 정밀세차/약식세차 횟수도 표시됩니다.

4 소모품·경정비 기록

해당 차량의 소모품/경정비/사고수리 기록을 확인할 수 있습니다.

7. 자주 묻는 질문

Q: 사진이 등록되지 않아요

A: 외관 사진은 전면/우측/후면/좌측 4장 모두 등록해야 합니다. 내부 사진도 1장 이상 등록해야 저장 버튼이 활성화됩니다.

Q: 담당자 이름이 목록에 없어요

A: 직접 이름을 입력하고 "추가" 버튼을 누르면 목록에 추가됩니다.

Q: 잘못 등록한 기록을 삭제하고 싶어요

A: 관리자 권한이 필요합니다. 관리자에게 문의해주세요.

Q: 차량 목록에서 특정 차량을 찾기 어려워요

A: 차량 목록 페이지에서 검색 기능을 이용하거나, 대시보드의 통계 카드를 클릭하면 해당 상태의 차량만 필터링됩니다.
